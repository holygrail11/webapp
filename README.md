# WebApp — Django Discord Clone

A simple Discord-like chat application built with Django. This is my first web application — a lightweight project for learning Django, authentication, and building a chat-style UI.

## Features
- User registration and authentication
- Channels/rooms to organize conversations
- Send and view messages in channels
- User profiles (basic)
- Responsive HTML/CSS frontend

> Note: This repo is a learning project, not intended for production use.

## Tech stack
- Python (Django)
- HTML, CSS
- (Optional) JavaScript for interactivity
